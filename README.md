## How to Use

To run Meetly locally, follow these steps:

- Clone the repository:
```bash
git clone https://github.com/thesunnymallick/Chat-Screen
```
- Install the dependencies:
```bash
cd meetly
npm install
```

- Start the app:

```bash
npm run dev
```
